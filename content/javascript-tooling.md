# JavaScript Tooling

## Near-production

- `now dev`
- `netlify dev -d public`

## Live reload

- `parcel public/index.html`
- `zero public`
- `eleventy --passthroughall --input=public --serve`

## Hot module reloading

- For parcel and zero, enable HMR using `module.hot && module.hot.accept();`
